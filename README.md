# V.R. GROUP TUITION Website

This repository contains a responsive single-page website for **V.R. GROUP TUITION**, an education institute based in Ahmedabad. The site serves as a modern promotional landing page showcasing tuition services, classes offered, fee details, highlights, and contact options. It is built using standard web technologies with a clean UI, smooth animations, and mobile-friendly design.

---

## Overview

The website is designed as a professional front-facing platform to help parents and students learn about the institute. It includes:

* A hero introduction section with branding
* Features and highlights of the tuition center
* Courses and standards covered (Nursery to 12th Commerce)
* Fee showcase cards
* Contact and inquiry options
* SEO meta tags and social preview meta support
* Fully responsive layout

---

## Features

* Responsive design for desktop and mobile
* Clean and modern visual layout
* Smooth scroll animations using AOS
* Bootstrap-based UI components
* Well-structured content sections
* Basic SEO meta information included
* Social media meta tags for sharing preview
* Icons and styled cards for better readability

---

## Technology Stack

* **HTML5**
* **CSS3**
* **Bootstrap**
* **JavaScript**
* **AOS (Animate on Scroll)**
* Images and static assets

No backend or database is used. The site is entirely static and front-end based.

## Running the Project

No build tools or server setup is required.

### Option 1: Open Directly

1. Download or clone the repository
2. Open `index.html` in any web browser

### Option 2: Serve Locally (Optional)

If you prefer running via a local server:

```bash
# Using Python 3
python -m http.server
```

Then open:

```
http://localhost:8000
```

---

## Usage

* Customize institute details directly in `index.html`
* Replace images if needed while keeping filenames or updating paths accordingly
* Update text sections such as features, courses, and fees as required
* SEO meta fields can be modified within the `<head>` section
